# Breakout Game

A classic Breakout game built with Phaser.js framework. This project is currently in early development stages.

## About

This is a browser-based implementation of the classic arcade game Breakout, where you control a paddle to bounce a ball and break bricks. The game is built using the Phaser 3 game framework.

## Features

- Ball physics with bouncing mechanics
- Paddle controls
- Brick destruction
- Score tracking
- Lives system
- Start button functionality

## Getting Started

### Prerequisites

A modern web browser (Chrome, Firefox, Safari, or Edge)

### Installation

1. Clone this repository:
```bash
git clone https://github.com/nicodevvv/breakout-game.git
cd breakout-game
```

2. Run a local server using one of the following methods:

#### Option 1: Using Live Server (VS Code Extension)
- Install the [Live Server extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code
- Right-click on `index.html` and select "Open with Live Server"
- The game will open in your default browser at `http://127.0.0.1:5500`

#### Option 2: Using Python 3
```bash
python3 -m http.server
```
- Open your browser and navigate to `http://localhost:8000`

### How to Play

- Use your mouse or keyboard to control the paddle
- Keep the ball in play by bouncing it off the paddle
- Break all the bricks to win
- Don't let the ball fall below the paddle!
